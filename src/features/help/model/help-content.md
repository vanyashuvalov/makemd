# Welcome to makemd

makemd is a markdown workspace for drafting, organizing, and exporting documents without losing your work between sessions.

## Quick start

- Create a new document from the sidebar.
- Edit markdown in the left panel.
- Read the formatted result in the preview panel.
- Use the document menu for rename, PDF export, copy markdown, and favorites.

## Favorites

- Save a document to Favorites from the document menu.
- Open the Favorites tab to reuse a saved snapshot as a new document.
- Favorites are available after sign-in.

## Sync

- Signed-in workspaces sync documents to Supabase.
- Local drafts stay in the browser as a quiet fallback.
- Cloud data loads first when both local and remote copies exist.

## Selection

- Hold Ctrl to select multiple documents.
- Use the bulk actions rail to delete, export, or copy markdown.

## Need more?

- Export the active document as PDF when you need a shareable version.
- Sign in with Google or email and password to keep your history available on this device.
